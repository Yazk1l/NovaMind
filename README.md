# NovaMind
NovaMind V2 adalah aplikasi belajar berbasis web untuk pelajar SMA/SMK dengan tema RPG pixel art. Fitur utamanya meliputi Quest Log, Notes, Flashcard, Focus Mode, Study Analytics, dan AI Nova Assistant. Pengguna dapat memperoleh XP, naik level, meraih achievement, serta membangun kebiasaan belajar yang lebih konsisten dan menyenangkan.
